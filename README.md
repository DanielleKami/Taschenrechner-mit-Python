# Taschenrechner mit Python

Ein **grafischer Taschenrechner**, implementiert in **Python**, der grundlegende sowie erweiterte mathematische Operationen unterstützt.  
Dieses Projekt dient als Übungsprojekt für GUI-Programmierung und mathematische Logik in Python.

## Funktionen

### Grundfunktionen
- Addition (+)
- Subtraktion (-)
- Multiplikation (*)
- Division (/)
- Modulo (%)
- Dezimalzahlen

### Erweiterte Funktionen
- Quadrat (x²)
- Quadratwurzel (√)
- Natürlicher Logarithmus (ln)
- Exponentialfunktion (e^x)
- Löschen einzelner Zeichen (`<-`)
- Komplettes Zurücksetzen (`C`)

### Weitere Features
- History-Anzeige für alle Berechnungen
- Fehlerbehandlung (z. B. Division oder Modulo durch 0)
- Automatische Rundung auf 5 Dezimalstellen
- GUI zentriert und immer im Vordergrund
